# Vera VS Code Extension

Basic syntax highlighting for the Vera programming language.

## Installation for Developers

1. Ensure you have `Node.js` installed.
2. Run `npm install` in this directory (or use `npx` directly in step 3).
3. Run `npx vsce package` to generate a `.vsix` file.
4. Install it in VS Code via:
   ```bash
   code --install-extension vera-0.1.0.vsix
   ```
